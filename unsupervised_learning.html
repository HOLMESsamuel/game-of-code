We use a kmeans model to clusterize our mails, trying from 2 to 5 clusters¶

In [120]:
from sklearn.cluster import KMeans
from sklearn import metrics

def clustering_algorithm(n_clusters, dataset):
    kmeans = KMeans(n_clusters=n_clusters, n_init=10, max_iter=300)
    labels = kmeans.fit_predict(dataset)
    silhouette = metrics.silhouette_score(dataset, labels, metric='euclidean')
    return silhouette

for i in range(2, 5):
    silhouette = clustering_algorithm(i, numerical_features)
    print(i, silhouette)

Conclusion¶

In this notebook we exploited the csv file of a dataset of mail, we manufactured some numerical features that we judged useful and tried to separate the data with a kmeans clustering model. The separation seams to work and we identified some features that separates the dataset better.
